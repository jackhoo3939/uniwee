## 📖 项目简介

UniWee官方网站是一个专为新加坡留学生设计的公寓租赁平台，提供优质的学生公寓信息和全方位的学生服务。

### ✨ 主要特色

- 🏠 **79个精选房源** - 覆盖新加坡主要高校周边
- 🎓 **按学校分类** - 10大学校分类，方便查找
- 📱 **响应式设计** - 完美支持PC、平板、手机
- 🎬 **视频展示** - 4个视频全方位展示公寓环境
- 🛠️ **管理后台** - 轻松管理网站内容
- 🚀 **快速部署** - 支持阿里云、腾讯云一键部署

## 📋 部署检查清单

部署前请确认以下项目：

### 服务器准备
- [ ] 已购买云服务器（阿里云/腾讯云）
- [ ] 已安装Nginx
- [ ] 已配置防火墙（开放80和443端口）
- [ ] 已配置域名解析

### 文件上传
- [ ] 所有HTML文件已上传
- [ ] CSS文件已上传

- [ ] 所有导航链接正常工作
- [ ] 视频可以正常播放
