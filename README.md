## ✨ 功能特点

### PC端（桌面版）
- 完整的导航菜单和页面布局
- 12个精选房源展示
- 4个社区生活视频
- 91个详细房源页面
- 中英文语言切换
- 关于我们和学生服务页面

### 移动端（响应式）
- 🍔 汉堡菜单导航
- 📞 固定底部联系栏
  - 品牌标语（左侧）
  - 电话号码（中间）
  - 微信二维码（右侧，可点击放大）
- 🎬 视频自动循环播放
- 📱 完美适配所有移动设备
